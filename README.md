# complex ai

## Running the project

### Frontend

Use the following on any OS:

```bash
cd frontend
npm i
node .
```

### Backend

Use the following on any OS:

```bash
cd backend
python -m venv .venv
source .venv/bin/activate # Or on Windows, ".venv/Scripts/activate.bat"
pip install .
# ... something else will be here but I  don't know how to start a Django
# application yet.... :,(
```
